jobcount.py
===========

GridEngine で各ユーザーの専有コア数 (r, qw をそれぞれ) を数えて表示する


    $ python jobcount.py
    
    *: r, -: qw
    
    ********************************************************************************-------------------------- Foo : 80 (26)
    ****************************************************************---------------- Bar : 64 (16)
    -------- Hoge : 0 (8)

    total : 144
